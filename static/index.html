API usage

Direct upload

POST /upload?fileName=<name>
Content-Type: <mime>
body: file bytes

Resumable upload

1. POST /upload/init  → { uploadId, fileName, contentType, uploadedBytes, progressUrl }
2. POST /upload/:uploadId  (repeat with remaining bytes) → { uploadId, uploadedBytes, partsUploaded }
3. POST /upload/:uploadId/complete  → { fileId, fileSize, contentType, url, ... }

Files

GET    /files                  → { total, offset, limit, items: [metadata] }
GET    /files/:id               → metadata
DELETE /files/:id
GET    /file/:id                (download, supports Range)
GET    /file/:id/:name          (with original name)

Health

GET /health → { status: "ok", redis: true, uptime }
