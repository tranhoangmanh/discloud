API reference

Direct upload

Resumable upload

POST /upload/init               → { uploadId, progressUrl }
POST /upload/:uploadId          (append more bytes)
POST /upload/:uploadId/complete → { fileId, ... }
GET  /upload/:uploadId/events   (SSE progress stream)

Files

GET    /files
GET    /files/:id
DELETE /files/:id
GET    /file/:id          (download)
GET    /file/:id/:name    (with original name)
