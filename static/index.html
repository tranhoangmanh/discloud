API usage

Direct upload

POST /upload?fileName=<name>
body: file bytes

Resumable upload

1. POST /upload/init  → { uploadId, progressUrl }
2. POST /upload/:uploadId  (repeat with remaining bytes)
3. POST /upload/:uploadId/complete  → { fileId, ... }

Files

GET    /files
GET    /files/:id
DELETE /files/:id
GET    /file/:id          (download)
GET    /file/:id/:name    (with original name)
